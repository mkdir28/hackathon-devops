# Prompt library

Example prompts for this job-match demo app and for external LLM workflows (CV analysis, career planning, job search).

## Folders

| Folder | Purpose |
|--------|---------|
| [`roles/`](roles/) | 30 ready-to-use job search queries by position (paste into the app search box) |
| [`cv/`](cv/) | CV / resume analysis and career planning prompts (placeholders) |
| [`job-search/`](job-search/) | Active job search planning prompts (placeholders) |

## Using role prompts in the app

1. Upload your CV (optional).
2. Open a file under `prompts/roles/`.
3. Copy the **Job search query** block into the search field.
4. Replace `{{PLACEHOLDERS}}` with your details (country, seniority, salary, stack).

## Placeholder convention

| Placeholder | Example |
|-------------|---------|
| `{{NAME}}` | Alex Chen |
| `{{SCHOOL}}` | UC Berkeley |
| `{{YEAR}}` | sophomore / 2027 |
| `{{COUNTRY}}` | United States |
| `{{CITY}}` | San Francisco |
| `{{SENIORITY}}` | Senior |
| `{{YEARS_EXPERIENCE}}` | 5 |
| `{{SALARY_MIN}}` | 8000 |
| `{{SALARY_MAX}}` | 12000 |
| `{{COMPANY_TYPE}}` | fintech startup |
| `{{CAREERS_URL}}` | https://example.com/careers |
